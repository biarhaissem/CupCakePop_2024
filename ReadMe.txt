CupCakePop - Front-End - Dev - Com Framework Angular

Este projeto foi gerado a partir do Angular CLI v13.3.2, utilizando o Bootstrap v5.1.3 e Bootstrap Icons v1.8.3 para customizações visuais.
Fizemos um site primeiramente sem framework e o reutilizamos para entender as diferenças nos dois procedimentos distintos.

O site é uma lojinha de cupcakes.
Contém cinco páginas, sendo uma delas, um formulário com 10 campos.

Como trabalhamos:

Instalamos a versão mais recente do Node.js.
Abrimos um CMD e utilize o comando npm install -g @angular/cli.
Fizemos download dos arquivos deste projeto.
Na pasta do projeto no CMD, utilizamos o comando npm install para instalar todas as dependências.
Ainda na pasta do projeto, utilizamos o comando ng serve --open no CMD para inicializar o projeto.
